# Great Bay Financial

This is a React project bootstrapped with Vite.

## Getting Started

To preview and work on this website locally, follow these steps:

### 1. Install Dependencies

First, ensure you have Node.js installed. Then, install the project dependencies by running:

```bash
npm install
```

### 2. Run the Development Server

To start the local development server with hot-reload (so you can see changes instantly):

```bash
npm run dev
```

This will start the server, typically accessible at [http://localhost:5173](http://localhost:5173) in your browser.

### 3. Build for Production

To create a production-ready build of the site:

```bash
npm run build
```

### 4. Preview the Production Build

To preview the built production site locally:

```bash
npm run preview
```
